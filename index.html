06 / How the System Learns
Peach Pilot — Internal

Three patterns that make agents smarter over time.

The platform is not static. Every action generates a learning signal. Every successful pattern crystallizes into reusable knowledge.

Pattern 01

Hermes

The reinforcement learning loop

Every agent action produces an outcome. Every outcome produces a reward signal. Every signal updates the agent's behavior on the next heartbeat. Agents get better through repetition.

Human approval
0.4

0.3

Pattern 02

Paperclip

Inter-agent coordination

Agents collaborate through structured messages, not ad-hoc chat. Five canonical schemas govern the conversation. Coordination is deterministic and auditable.

01Status update
02Escalation
03Approval request
04Governance violation
05Collaboration query
Pattern 03

Autoresearch

Recursive self-improvement

Agents hypothesize improvements to their own behavior, backtest against the knowledge graph, and propose changes to their manager. Learning is continuous but governed.

→Hypothesize an improvement
→Backtest against historical cases
→Propose to manager for approval
→Deploy and monitor 20 live actions
Skill crystallization

When an agent's approach succeeds on 20+ instances with an 80%+ approval rate, the pattern is extracted into a reusable skill. Other agents inherit it. Cross-department skills get promoted by the Knowledge Curator.

Day one agents are adequate. Day 180 agents are expert. The system compounds.
